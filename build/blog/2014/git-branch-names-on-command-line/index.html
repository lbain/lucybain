Git Branch Names on Command Prompt

I often forget which branch I'm on. Sure, I have git branch aliased (it's probably one of my most frequently used commands) but sometimes I know I'm on another branch. A few commits and merges later I find out I was wrong, and it's time for some branch cleaning.

Recently my coworker Simon gave me the following code: 

parse_git_branch() {
   git branch 2> /dev/null | sed -e '/^[^*]/d' -e 's/* \(.*\)/ (\1)/'
}
export PS1="\W\[\033[32m\]\$(parse_git_branch)\[\033[00m\] $ "

This code should go in your ~/.bash_profile. Once you've added it, you'll need to source with $ source ~/.bash_profile (or $ . ~/.bash_profile if you want to save a few key strokes!).
